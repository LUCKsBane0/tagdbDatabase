# tagdbDatabase
## Little tag-Database for the terminal
#### Usage
Print help message:

    tagdb -h
    
Add tag to database:
 
    tagdb -a <tag> <content>

Search for tags in database:

    tagdb -q <tag> -q <tag>
    
Delete tag with ALL content:
    
    tagdb -d <tag>
